How Swift Tabs Works

Key Combinations

  • Primary Key (Alt): Tab navigation & history
  • Secondary Key (Ctrl): Window management
  • Tertiary Key (Shift): Modifies actions

Tab Navigation

Primary Key + Vertical Swipe

Switch between adjacent tabs

Primary Key + Horizontal Swipe

Navigate tab history (back/forward)

Window Management

Secondary Key + Vertical Swipe

Switch between windows

Secondary + Tertiary + Swipe

Merge/split windows, consolidate tabs

Advanced Features

Tab Selection

Use Tertiary key to select multiple tabs

Tab Actions

Move, close, or create new windows from tabs
